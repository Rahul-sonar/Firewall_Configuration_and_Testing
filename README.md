# Firewall_Configuration_and_Testing

# Objective
To understand firewall concepts and perform firewall configuration and testing using UFW on Kali Linux.

# Tools Used
- UFW (Uncomplicated Firewall)
- Nmap

# Tasks Performed
- Checked firewall status
- Enabled UFW firewall
- Configured default deny and allow rules
- Allowed SSH (22) and HTTP (80)
- Blocked FTP port (21)
- Tested connectivity using Nmap
- Blocked a malicious IP address
- Enabled and analyzed firewall logs
- Verified final firewall configuration

# Repository Contents
- Firewall configuration and testing_Report.pdf
- screenshots/
- README.md

# Result
Demonstrated practical firewall management, rule configuration, traffic filtering, and logging for securing a Linux system.
